# 🎓 AI 기반 영어 시험 문제 자동 생성 시스템

> **고등학교 1학년 영어 내신 대비를 위한 End-to-End 문제 생성 파이프라인**
>
> 6,000+ 기출문제 DB × 10종 교과서 본문 × Gemini LLM → 새 문제 자동 생성 → Word 시험지 출력

### 프로젝트 배경

고등학교 영어 내신 시험 준비를 위해, 수능·모의고사·학교 기출문제를 체계적으로 수집·분류하고, 교과서 본문을 기반으로 AI가 새로운 시험 문제를 자동 생성하는 시스템을 개발했습니다.

단순한 문제 생성을 넘어, **출제 전략 분석 → 문제 생성 → 오답 선지 생성 → 학생 풀이 시뮬레이션 → 검수·재생성**의 5단계 품질 보증 파이프라인을 구축하여 실제 시험 수준의 문제 품질을 달성했습니다.

## 🔑 핵심 기능

### 1. 데이터 수집·파싱 파이프라인

#### 학교 기출문제 파싱 (`txt_parser.py` — 877줄)
- **473개 학교 시험 파일**에서 **6,061개 문제** 자동 추출
- 단원별 블록 분리 → 문제별 분할 → 지시문·지문·선지(①~⑤)·정답 구조화
- 객관식 5지선다 + 서술형/주관식 문제 모두 처리
- 15개 이상의 정규식 패턴으로 다양한 출제 형식 대응

#### 모의고사 PDF 파싱 (`pdf_parser.py` — 604줄)
- `pdfplumber`로 수능·모의고사 PDF 텍스트 추출
- 문제지(problem) + 해설지(solution) PDF 자동 매칭
- 해설지에서 정답표 추출: 정규식 → Gemini LLM 이중 파이프라인
- 섹션 헤더(`[31~34]`), 꼬리말 자동 제거, 연속 문제 처리

#### 교과서 본문 추출 (`textbook_parser.py` — 154줄)
- **10종 출판사** 교과서 PDF에서 영어 본문만 추출
- PDF 페이지의 왼쪽 62% 영역 크롭 (오른쪽은 한국어 해석)
- 단원 번호 → 시험 기간(중간/기말) 자동 매핑

#### 파일명 메타데이터 파서 (`filename_parser.py` — 188줄)
- `(개정)2018년_고1_1학기 기말_경복고등학교_서울시 종로구_영어_NE능률(양현권).txt`
- → 연도, 학년, 학기, 학교명, 지역, 출판사, 저자 자동 추출
- macOS NFD→NFC 유니코드 정규화 처리

**분류 대상 22개 유형:**

| 문장삽입 | 285 | 심경/분위기 | 77 |
| 제목 | 247 | 기타 | 52+ |

### 3. AI 문제 생성 엔진 (`question_generator.py` — 2,514줄)

프로젝트의 핵심으로, **5단계 품질 보증 파이프라인**을 통해 실제 시험 수준의 문제를 생성합니다.

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  Stage 1     │    │  Stage 2     │    │  Stage 3     │
│  분석 에이전트 │──▶│  생성 에이전트 │──▶│  오답 생성    │
│              │    │              │    │  에이전트     │
│ 교과서 본문 + │    │ 출제 전략 +   │    │              │
│ 기출 예시 분석│    │ 지문 기반     │    │ 매력적인     │
│ → 출제 전략  │    │ 문제 생성    │    │ 5지선다 생성  │
└──────────────┘    └──────────────┘    └──────┬───────┘
                                               │
┌──────────────┐    ┌──────────────┐           │
│  Stage 5     │    │  Stage 4     │           │
│  검수 에이전트 │◀──│  시뮬레이션   │◀──────────┘
│              │    │  에이전트     │
│ 품질 평가    │    │              │
│ 불합격 시    │    │ 학생이 정답   │
│ 자동 재생성  │    │ 없이 풀어봄  │
└──────────────┘    │ 정답 불일치·  │
                    │ 복수정답 감지 │
                    └──────────────┘
```

#### Stage 1: 분석 에이전트
- 교과서 본문 + 동일 유형 기출 예시 → 최적 출제 전략 수립
- 어떤 문장을 변형하고, 어떤 문법 포인트를 출제할지 계획

#### Stage 2: 생성 에이전트
- 출제 전략에 따라 실제 문제 생성
- 지문 가공 (빈칸·밑줄·순서 재배열 등)
- 22가지 유형별 특화 프롬프트

#### Stage 3: 오답 선지 생성 에이전트
- 정답을 포함한 5지선다 선지 생성 (별도 전문 에이전트)
- 유형별 전략: 빈칸추론은 의미적 오답, 어법은 문법적 오답 등
- 어법/어휘 유형은 `①②③④⑤` 자동 매핑 (LLM 호출 불필요)

#### Stage 4: 학생 풀이 시뮬레이션 에이전트
- LLM이 고1 학생 역할로 **정답 없이** 문제를 풀어봄
- 정답 불일치 감지 → 문제 결함 탐지
- 복수정답 가능성 감지 → 자동 재생성 트리거

#### Stage 5: 검수 에이전트
- 문제 품질 10점 만점 평가
- 불합격 문제 → 피드백 반영 자동 재생성 (최대 1회)
- 시뮬레이션 결과 반영한 엄격 검수

#### 출제 소스 지원
- **교과서 기반**: 교과서 본문 + 동일 유형 기출 예시 참조
- **모의고사 기반**: 수능/모의고사 지문을 새 유형으로 변형
  - 특정 문제 번호 선택 기능 (예: 21번, 29번, 33번만)

### 4. 웹 기반 관리 UI (`db_server.py` — 1,536줄)

Python 표준 라이브러리 `http.server`만으로 구현한 **프레임워크 무의존(Zero-dependency) 웹 UI**:

| 페이지 | 기능 |
|--------|------|
| 🏠 대시보드 | DB 통계, 문제 수, 유형 분포 |
| 📝 시험 목록 | 479개 시험 필터링·검색 |
| 🔍 문제 검색 | 키워드 기반 전문 검색 |
| 📊 유형 분석 | 22개 유형별 분포 시각화 |
| 📚 교과서 뷰어 | 10종 교과서 본문 열람 |
| 📋 모의고사 뷰어 | 모의고사 문제 상세 보기 |
| ✨ **AI 생성** | 유형·수량·소스 선택 → 실시간 생성 → Word 다운로드 |

#### AI 문제 생성 UI 상세
- 출제 유형 선택 (22종)
- 출제 형식 선택 (객관식/서술형/혼합)
- 소스 선택 (교과서 or 모의고사)
- 모의고사 고급 옵션: 특정 문제 번호 체크박스 선택
- 비동기 생성 + 실시간 진행 상태 표시
- 생성 결과 메타데이터: 검수 결과, 시뮬레이션 결과, 재생성 여부

### 5. Word 시험지 자동 출력 (`word_output.py` — 636줄)

- **표지**: 시험 제목, 학교명, 학년, 날짜
- **객관식 영역**: 문제번호, 지시문, 지문(밑줄 포맷), 5지선다
- **서술형 영역**: 지시문, 조건, 지문, 답안란
- **정답 및 해설**: 정답표 + 해설
- `<u>` HTML 태그 → Word 밑줄 자동 변환
- 한글 폰트(나눔고딕) 적용

## 💡 기술적 도전과 해결

### 1. 다양한 시험 문제 형식 파싱
**문제**: 473개 학교의 시험 파일이 모두 다른 형식 — 단원 표기, 선지 표기, 정답 표기가 제각각
**해결**: 15개 이상의 정규식 패턴을 단계적으로 적용하는 파이프라인 설계. 단원 블록 → 문제 블록 → 세부 파싱의 3단계 분리로 유연하게 대응

### 2. PDF 정답표 추출의 불확실성
**문제**: 모의고사 해설 PDF의 정답표가 표 형식, 텍스트 나열 등 다양한 형태
**해결**: 정규식 기반 추출을 1차로 시도하고, 실패 시 Gemini LLM에게 이미지를 포함한 프롬프트로 2차 추출. 이중 안전장치로 추출 성공률 극대화

### 3. LLM 생성 문제의 품질 보증
**문제**: LLM이 생성한 문제가 복수정답, 정답 불명확, 오답 매력도 부족 등의 결함 가능
**해결**: 5단계 멀티 에이전트 파이프라인으로 해결
  - 정답 없이 학생 시뮬레이션 → 복수정답 감지 시 자동 재생성
  - 10점 만점 검수 → 불합격 시 피드백 포함 재생성
  - 어법/어휘 유형은 선지를 규칙 기반으로 자동 매핑하여 LLM 오류 방지

### 4. 교과서 PDF에서 영어만 추출
**문제**: 교과서 PDF가 좌측 영어 본문 + 우측 한국어 해석의 2단 레이아웃
**해결**: `pdfplumber`의 page cropping 기능으로 좌측 62%만 크롭하여 영어 텍스트만 정확 추출

### 5. macOS 유니코드 이슈
**문제**: macOS 파일시스템이 한글 파일명을 NFD(분해형)로 저장하여 DB 조회 불일치
**해결**: 모든 파일명 처리 시 `unicodedata.normalize('NFC', ...)` 적용

### 6. 웹 프레임워크 없는 UI 구현
**문제**: Flask/Django 등 외부 의존성 없이 즉시 실행 가능한 웹 UI 필요
**해결**: Python 표준 라이브러리 `http.server`만으로 라우팅, API, 비동기 생성 관리, 반응형 CSS/JS까지 구현

python db_server.py
